How to fire a command

prefix: {{ data.prefix }}
Speak the prefix word, then a command. Examples:

{{ data.prefix }} scroll down     → Page Down

Command prefix

The wake word you say before every command (Action Mode uses it too).
3+ letters, no spaces, not a common English word. Takes effect on your next command — no restart.
