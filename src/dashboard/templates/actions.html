How to fire an action

prefix: {{ data.prefix }}
{% if data.require_prefix %}
Say the prefix word, then an action. Examples:
{{ data.prefix }}, open notepad              → launches a configured app
{{ data.prefix }} search the web for cats  → opens a web search
{{ data.prefix }}, open downloads folder    → opens a folder
{{ data.prefix }}, take a note that …      → saves a note
{% else %}
Just say it — no prefix needed. A shortcut fires only when it resolves; anything else types normally. Examples:
open spotify            → launches a configured app
go to github.com        → opens a site
open downloads folder   → opens a folder
search the web for cats → opens a web search
Media, notes, and events still want the prefix (“{{ data.prefix }}”) so everyday words aren’t caught.
{% endif %}
